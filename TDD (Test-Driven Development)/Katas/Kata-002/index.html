Wizard de la Kata 002

Recorreremos la kata paso a paso: definir la regla, ejecutarla, ver resultados y generar el código de la calculadora.

2. Ejecutar la kata

Paso 1 — Definir la kata

La kata consiste en convertir los números del 0 al 100 aplicando estas reglas:

  • Si es múltiplo de 3, mostrar Fizz.
  • Si es múltiplo de 5, mostrar Buzz.
  • Si es múltiplo de 3 y 5, mostrar FizzBuzz.
  • En cualquier otro caso, mostrar el número.

Paso 2 — Ejecutar la kata

Haz clic para ejecutar la lógica de FizzBuzz y generar la secuencia.

Paso 3 — Ver resultados en formato tabla

Los resultados se muestran como una tabla con estado visual tipo TDD: verde para acierto y rojo para fallo.

#EsperadoObtenidoEstado
Fase roja: se define una prueba nueva y se comprueba que falla, lo que confirma que la regla no está implementada todavía.
Fase verde: se implementa la mínima lógica necesaria para que la prueba pase y la kata funcione.
Fase de refactorización: se mejora el código manteniendo el mismo comportamiento y dejando la solución más limpia.
